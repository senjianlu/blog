在 Mac 启动时自动运行脚本

前言

最近将大量基础设置迁移到了 Mac mini 上，有些应用通过安装可以简单地设置开机自启，但是不少脚本和服务还是需要手动运行。
为了方便管理和使用，决定在 Mac 启动时自动运行一些脚本。

  1. 建立脚本文件
  2. 修改脚本的权限
  3. 将打开方式修改为终端

使用 Docker 部署 GOST 并开启 HTTPS 代理

前言

HTTP 和 SOCKS5 协议由于没有加密，所以在使用时会有一定的风险，而 HTTPS 协议则是一种加密的协议，可以有效的保护数据的安全性。本文将介绍如何使用 Docker 部署 GOST 并开启 HTTPS 代理。

方案概述

  1. 安装 Docker 环境
  2. 使用 Docker 部署 GOST 并开启 HTTP 代理
  3. 使用 acme.sh 申请证书
  4. 将证书挂载到 GOST 容器中并开启 HTTPS 代理

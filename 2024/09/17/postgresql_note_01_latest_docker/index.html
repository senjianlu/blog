前言

数据库最好跑在数台单独的虚拟机上来确保高可用。
不过我的边缘业务并没有那么高需求，即使所有服务器都 Down 了也有 Cloudflare 代理业务用的服务器做最后一层保障，因此怎么方便怎么来吧。

二、确定容器运行参数

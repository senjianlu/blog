使用 Docker 部署 Loki + Grafana 日志聚合系统

前言

尝试使用 Loki + Grafana 替换掉和我业务量完全不匹配的 ELK。

方案概述

  1. 安装 Docker 环境
  2. 使用官方的 docker-compose.yaml 启动 Loki + Grafana + Promtail
  3. Grafana 配置 Loki 数据源
  4. 配置 Nginx 反代 Loki 并添加认证功能
  5. 配置 Docker 插件来测试日志推送 Loki 功能

  • Loki：负责存储日志和处理查询。
  • Promtail：代理，负责收集日志并将其发送给 Loki。
  • Grafana：用于日志展示等。

如果需要收集单独运行的 Docker 容器的日志的话，官方也有插件：Docker driver client
